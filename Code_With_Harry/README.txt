JS Minimum Core Concepts Before Starting React:-

BASIC:
- var, let, const, primitive data types & strings
- conditional, loop, for each loop

INTERMEDIATE:
- Arrow functions vs normal functions
- Lexical this
- Events, Handling events
- Spread operator, Destructuring
- named imports, default import

ADVANCED:
- asynchronus nature, call back system, promises
- async, await
- map, filter, reduce

REACT CONCEPTS:
- Hooks, components
- Socket
- Web RTC

NOTES:
- don't use innerHTML for security reasons instead use textContent
- if you can create a UI library then you are good in JavaScript.
- Just learn how to use webpack, babel, ES6 and a few more things.
